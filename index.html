操作方式

Operation Methods

用户可通过“系统设置->键盘->键盘快捷键->服务->通用选择”启用右键Services功能，便捷地对选定文件执行压缩、解压缩操作。同时，将文件拖曳至应用界面，就能轻松完成压缩或解压缩任务，操作流程简洁明了，提升文件处理效率。
Users can enable the right - click Services function through "System Settings -> Keyboard -> Keyboard Shortcuts -> Services -> General Selection" to conveniently perform compression and decompression operations on selected files. At the same time, by dragging files to the application interface, compression or decompression tasks can be easily completed. The operation process is simple and clear, improving file processing efficiency.
  1. 右键服务：通过系统设置 -> 键盘 -> 服务 启用
  2. Right - click service: Enable through System Settings -> Keyboard -> Services
  3. 拖拽操作：直接将文件拖入应用窗口
  4. Drag and drop operation: Directly drag files into the application window
